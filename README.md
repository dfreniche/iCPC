iCPC
=======

Amstrad emulator for iPhone/iPad.

Based on CrocoDS by Miguel Vanhove
Based on Win-CPC by Ludovic Deplanque.

How to use ?
============

Open the .xcode file and run it to your iPhone/iPad :)
Image disks (.dsk) have to be send via iTunes
